# Welcome to LinkUni üéì  
**Discover. Share. Collaborate.**

LinkUni is a dynamic platform crafted exclusively for university students to effortlessly share, discover, and organize educational resources. Whether it's previous year questions (PYQs), notes, research papers, or project files, LinkUni connects university communities, making learning and collaboration seamless.

## üöÄ Key Features  
- **Seamless Resource Sharing**  
  Share academic materials like PYQs, notes, projects, and research papers with peers, bridging knowledge gaps in your community.

- **Personalized Dashboards**  
  Manage your uploads, track downloads, and save valuable resources in your own customized space. Showcase your expertise through your unique profile.  

- **AI-Powered Insights** *(Coming Soon)*  
  Prepare smarter with AI-driven summaries and interactive chats for uploaded resources.  

- **Resource Rating & Feedback**  
  Help others find the best materials by rating and reviewing shared content.

- **Custom Collections** *(Coming Soon)*  
  Organize your favorite resources into collections and collaborate with peers to build a rich repository of academic knowledge.

- **Enhanced Discoverability**  
  Navigate a well-categorized library to find exactly what you need, with tools like starred top picks and advanced filtering.

## üåü Our Mission  
LinkUni empowers students by fostering a culture of sharing, collaboration, and easy access to educational resources. It's not just a platform; it's your academic ally.

## üöÄ How to Set Up Locally  

Follow this guide to get LinkUni running on your local machine and contribute to its development:  

### Prerequisites  
- Node.js (v16+ recommended)  
- npm or Yarn  
- Firebase app credentials for authentication (optional for basic setup)

### Steps  
1. **Clone the Repository**  
   ```bash  
   git clone https://github.com/chhavikant-01/link-uni.git
   ```  

2. **Install Dependencies**  
   ```bash  
   npm install  
   ```  

3. **Set Up Environment Variables**  
   Create a `.env` file with the following:  
   ```env
   REACT_APP_BASE_URL=
   REACT_APP_API_KEY=
   REACT_APP_AUTH_DOMAIN=
   REACT_APP_PROJECT_ID=
   REACT_APP_STORAGE_BUCKET=
   REACT_APP_MESSAGING_SENDER_ID=
   REACT_APP_APP_ID=
   REACT_APP_MEASUREMENT_ID=
   REACT_APP_ALLOWED_EMAIL_DOMAIN=
   ```  

4. **Start the Server**  
   ```bash  
   npm run start
   ```  

5. **Access the App**  
   Open your browser and navigate to `http://localhost:3000`.

6. **Setup Backend**  
Please refer [this](https://github.com/chhavikant-01/link-uni-backend)
